## 강북에서 만날 수 있는 고양이 카페

### (1) 이대 커피볶는 사진관

카페 이름에 고양이가 빠졌다 해서 카페에 고양이가 없는 것은 아니죠.
요즘은 카페를 운영하는 사장님이 키우시는 고양이들이 함께하는 공간도 늘어나는 추세입니다. 바로 그런 카페 중 하나인데요.
